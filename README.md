Visit us at: [UBJ Houseboats](https://ubjhouseboats.com)

## 📋 About

UBJ Houseboats offers authentic Kerala backwater experiences with:
- Day cruises and overnight stays
- Traditional Kerala cuisine
- Experienced crew and comfortable cabins
- Scenic routes through Punnamada Kayal

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js (Static file server)
- **Deployment**: Vercel
- **Performance**: Optimized for mobile and desktop

## 🚀 Local Development

1. Clone the repository:
```bash
git clone <your-repo-url>
cd ubj-houseboats
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and visit:
```
http://localhost:3000
```

## 📤 Deployment

### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel
```

3. Follow the prompts to complete deployment

### Deploy via GitHub

1. Push your code to GitHub
2. Import the repository in Vercel dashboard
3. Vercel will automatically deploy
